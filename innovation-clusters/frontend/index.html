What Are Innovation Clusters?

History shows us that major innovations don't happen randomly—they cluster together in waves. This analysis examines FDA drug approvals and USPTO breakthrough patents from 1920–2024 to identify these temporal patterns.

Using statistical clustering analysis (>2σ above mean), we reveal which time periods experienced unusual spikes in innovation across pharmaceuticals, semiconductors, telecommunications, and computing.

Methodology

Data Sources

  • FDA Orange Book API (drug approvals, 1938-2024)
  • USPTO PatentsView API (breakthrough patents, 1920-2024)
  • Focus on high-citation patents (top 10%)

Clustering Method

  • Statistical threshold: >2σ above mean
  • Minimum cluster size: 3 consecutive years
  • Identifies periods of unusual innovation density

Technology Categories

  • Pharmaceuticals (CPC: A61K)
  • Semiconductors (CPC: H01L)
  • Telecommunications (CPC: H04)
  • Computing (CPC: G06F)

Research Context

This analysis reveals temporal patterns in innovation clustering but does not establish causality. Multiple factors—economic conditions, policy changes, technological dependencies, and research funding—may contribute to these patterns.

Data sources: FDA Orange Book API | USPTO PatentsView API

Analysis method: Statistical temporal clustering (>2σ threshold)
